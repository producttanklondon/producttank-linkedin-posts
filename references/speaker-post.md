# Speaker introduction post

**Timing:** one per speaker — ~15, ~10, and ~5 days before the event.
**Goal:** introduce a single speaker and their talk, build anticipation.

Each post is numbered in the header: `(1/3)`, `(2/3)`, `(3/3)`.

## Structure

1. Bold header line wrapped in 📣 with the speaker number, e.g.
   `📣 𝗜𝗻𝘁𝗿𝗼𝗱𝘂𝗰𝗶𝗻𝗴 𝗼𝘂𝗿 𝗳𝗶𝗿𝘀𝘁 𝘀𝗽𝗲𝗮𝗸𝗲𝗿 𝗳𝗼𝗿 𝗼𝘂𝗿 𝗻𝗲𝘅𝘁 𝗲𝘃𝗲𝗻𝘁 (1/3) 📣`
2. 👤 Speaker name (tagged + linked) and a short bio (role, company, background).
3. 🎤 Talk title and a 2–3 sentence description of what attendees will learn.
4. Detail block (date + time).
5. Note if the event can be enjoyed online or in person.
6. `RSVP now 👇` + `[meetup URL]`.

## Template

```
📣 𝗜𝗻𝘁𝗿𝗼𝗱𝘂𝗰𝗶𝗻𝗴 𝗼𝘂𝗿 <first/second/third> 𝘀𝗽𝗲𝗮𝗸𝗲𝗿 𝗳𝗼𝗿 𝗼𝘂𝗿 𝗻𝗲𝘅𝘁 𝗲𝘃𝗲𝗻𝘁 (<n>/3) 📣

👤 @Speaker Name:
<1–2 sentence bio — role, company, notable background.>

🎤 <Speaker first name>'s talk - <Talk title>:

```
