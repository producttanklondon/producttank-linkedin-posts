# Event reminder post

**Timing:** ~2–3 days before the event.
**Goal:** a short, warm nudge to drive last-minute RSVPs and attendance.

Keep this one brief — it's a reminder, not a full announcement.

## Structure

1. Bold header line wrapped in 📣, e.g. `📣 𝗣𝗿𝗼𝗱𝘂𝗰𝘁𝗧𝗮𝗻𝗸 𝗟𝗼𝗻𝗱𝗼𝗻 𝗲𝘃𝗲𝗻𝘁 𝗿𝗲𝗺𝗶𝗻𝗱𝗲𝗿 📣`
2. A one-line warm reminder referencing the day and the theme.
3. Thank the sponsor (tag them).
4. Detail block (date + time).
5. Optional: `RSVP now 👇` + `[meetup URL]`.

## Template

```
📣 𝗣𝗿𝗼𝗱𝘂𝗰𝘁𝗧𝗮𝗻𝗸 𝗟𝗼𝗻𝗱𝗼𝗻 𝗲𝘃𝗲𝗻𝘁 𝗿𝗲𝗺𝗶𝗻𝗱𝗲𝗿 📣

We hope to see you all this <weekday> to hear from Product Leaders and Product
Managers about <theme>!

Thank you to our event sponsor: @Sponsor

🗓️ Date: <Month Day>
🕑 Time: 7pm
```
