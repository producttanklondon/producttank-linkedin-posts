# ProductTank London — LinkedIn Posts

Drafts LinkedIn posts for ProductTank London events on the official
[company page](https://www.linkedin.com/company/producttanklondon/), matching the
established brand voice and posting cadence.

## When to use this

Use this skill whenever someone needs to write LinkedIn content for a ProductTank
London meetup. There are four post types:

1. **Event announcement** — the headline post for the event
2. **Speaker introduction** — one per speaker, numbered (1/3, 2/3, 3/3)
3. **Event reminder** — a short nudge a few days before
4. **Post-event recap** — a thank-you / wrap-up after the event

## Posting cadence

Posts are created and scheduled on the LinkedIn page **after the event is published
on Meetup**. The historical schedule:

| Post | Timing |
| --- | --- |
| Announcement | When the Meetup event is published |
| Introducing speaker #1 | ~15 days before the event |
| Introducing speaker #2 | ~10 days before the event |
| Introducing speaker #3 | ~5 days before the event |
| Event reminder | ~2–3 days before the event |
| Post-event recap | 1–2 days after the event |

## Universal rules (apply to every post)

- **Tag people and companies.** Wherever possible, tag the individuals named and
  their company pages (if they exist) — both in the **text** of the post and in the
  **image**. Speakers, the MC/moderator, and the sponsor should all be tagged.
- **Always thank the sponsor** when there is one.
- **Headers use bold styling.** ProductTank posts open with a line wrapped in
  📣 emoji and rendered in mathematical-bold Unicode characters (e.g.
  𝗣𝗿𝗼𝗱𝘂𝗰𝘁𝗧𝗮𝗻𝗸 𝗟𝗼𝗻𝗱𝗼𝗻). LinkedIn has no rich-text formatting, so this
  Unicode trick is how posts get "bold" headers. Convert the header line to bold
  Unicode before posting.
- **Standardised detail block** near the end of most posts:
  ```
  🗓️ Date: <Month Day>
  🕑 Time: 7 pm (UK Time)
  ```
- **Close with a call to action** and the Meetup RSVP link: `RSVP now 👇` followed
  by the `[meetup URL]`.
- Note whether the event is **in person, online, or hybrid** when relevant.

## How to draft a post

1. Identify which of the four post types is needed.
2. Read the matching template in [references/](references/):
   - [references/announcement-post.md](references/announcement-post.md)
   - [references/speaker-post.md](references/speaker-post.md)
   - [references/event-reminder-post.md](references/event-reminder-post.md)
   - [references/post-event-recap.md](references/post-event-recap.md)
3. Fill in the event specifics (date, venue, speakers, talk titles, sponsor, MC).
4. Apply the universal rules above (tagging, bold header, sponsor thanks, RSVP CTA).
5. Produce the final post text, ready to paste/schedule on LinkedIn. Where a name
   should be tagged, mark it clearly (e.g. `@Speaker Name`) so the poster can
   complete the tag in LinkedIn's editor.
